🆘 TROUBLESHOOTING:

Issue: "npm command not found"
→ Install Node.js from https://nodejs.org/

Issue: Port 5000 already in use
→ Change PORT in backend/.env or stop other app

Issue: Blank white screen
→ Press Ctrl+Shift+R (hard refresh) or check F12 console

Issue: Form won't submit
→ Check F12 console for errors, ensure both servers running
